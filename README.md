# Grading System

This is a simple Python program that takes a student's score (0–100) and returns the corresponding grade based on a standard school grading scale.

## Grading Scale:
- 70–100 → A
- 60–69 → B
- 50–59 → C
- 45–49 → D
- 40–44 → E
- 0–39  → F

## How to Use
1. Run the program using Python 3:
   ```bash
   python grading_system.py
